# Retail-Sales-Analysis
This project focuses on analyzing retail sales data using Microsoft Excel. The goal is to transform raw sales data into meaningful business insights through data cleaning, calculations, pivot tables, and charts. The project demonstrates practical Excel skills commonly required for data analyst internships and entry-level roles.
Objectives

Analyze total revenue and sales performance

Compare sales across regions and products

Understand customer payment behavior

Create clear and professional visualizations

Practice real-world Excel data analysis workflow
